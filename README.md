Personal study materials for the AWS Certified Advanced Networking - Specialty exam.

## Exam Overview

- **Code**: ANS-C01
- **Questions**: 65 (50 scored + 15 unscored)
- **Time**: 170 minutes
- **Passing Score**: 750/1000
- **Cost**: $300 USD

### Domain Breakdown

| Domain | Weight |
|--------|--------|
| Network Design | 30% |
| Network Implementation | 26% |
| Network Security, Compliance & Governance | 24% |
| Network Management & Operations | 20% |

## Topics Covered

- VPC architecture, CIDR planning, subnetting
- Direct Connect (dedicated/hosted, VIFs, DX Gateway, LAGs, MACsec)
- Site-to-Site VPN (IKE, IPsec, tunnel options, ECMP)
- Transit Gateway (attachments, route tables, peering)
- Hybrid DNS (Route 53 Resolver, inbound/outbound endpoints)
- Load balancing (ALB, NLB, GWLB)
- CloudFront & Global Accelerator
- VPC Endpoints & PrivateLink
- Network security (Network Firewall, WAF, Shield)
- Monitoring & troubleshooting (Flow Logs, Traffic Mirroring, Reachability Analyzer)
